# Minecraft 1.12.2 Nostalgic Chill Tech Pack

This repository tracks the source/config/customization files for a Minecraft 1.12.2 nostalgic chill tech pack built for PrismLauncher.

The live pack is edited directly inside the PrismLauncher instance `minecraft/` folder. Git intentionally tracks only selected scripts, resource packs, and curated configuration files so changes show up immediately without committing runtime data.

Mod `.jar` files are intentionally not committed. Installable exports will be published through GitHub Releases.

## Tracked Sources

- CraftTweaker scripts in `scripts/`
- Required/custom resource packs in `resourcepacks/`
- Selected pack-defining config files under `config/`
- Custom CoFH/Thermal progression content

## Not Tracked

- Mod jars
- Saves and backups
- Logs and crash reports
- Screenshots
- Launcher/user-specific files
- Runtime caches
