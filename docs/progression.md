# Progression Notes

This pack aims for a nostalgic, chill Minecraft 1.12.2 tech progression.

## Current Direction

- IndustrialCraft 2 is the early backbone.
- Thermal Expansion and TeslaCoreLib components are gated behind IC2-oriented infrastructure.
- Industrial Foregoing is a gated automation layer rather than an early all-in-one shortcut.
- Draconic Evolution and GraviSuite tools are treated as late-game milestones.

## Tracked Balance Sources

- `scripts/`
- `config/brandon3055/CustomFusionRecipes.json`
- selected worldgen and mod config files
- custom Thermal/CoFH content files
